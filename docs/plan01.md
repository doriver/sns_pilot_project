# 작은 SNS
## 주요기능
### 회원
* 프로필 이미지, 닉네임
* JWT 로그인
* 권한 - 일반 유저, 인플루언서, 관리자
### 게시판
* 이미지 첨부 글 + 조회수 + 댓글 + 좋아요
* 프론트에서 보여질 형식
  * 타임라인 형식 - 메인화면에 조회수, 작성날짜 기준으로 10개만 보여줄것 
  * 글 목록 + 글 디테일 - 페이지네이션

### 단체 채팅
* 단체 채팅방 생성, 종료
  * 인플루언서만 채팅방 생성 가능 - 채팅방 이름, 제한인원 등 입력하고 생성
  * 채팅방 만든사람만 종료 가능
* 단체 채팅방 목록
  * 방마다 현재 참석자수, 제한인원 표시
* 단체 채팅방
  * 현재 참석중인 사용자들 목록


## 기술스택
### 프론트
* 바닐라 JS
### 백엔드
* Express.js
* MongoDB
